## Description

Neo Bug Forge is a VS Code extension that fixes your bugs instantly. Paste broken code, hit **Fix My Bug**, and get back a corrected version with a plain-English explanation, unified diff, and a unit test — all in one shot.

No tab-switching. No copy-pasting into ChatGPT. No guessing what went wrong.

**How it works:**
1. Open the Neo Bug Forge panel in VS Code
2. Paste your broken code (or select it with Ctrl+Shift+F)
3. Optionally paste the error message
4. Hit Fix My Bug → get fixed code, explanation, diff, and test case in seconds

**What you get back:**
- ✅ Fixed code (complete, ready to paste)
- ✅ Root cause label (IndexError, TypeError, logic_error…)
- ✅ Confidence score (0–100%)
- ✅ Unified diff (so you can see exactly what changed)
- ✅ Minimal unit test that would have caught the bug

**Supported languages:** Python, JavaScript, TypeScript, Rust, Go, Java, C++, and more (AUTO mode detects the language for you)

**Free tier:** 100 fixes/month, no credit card required.
